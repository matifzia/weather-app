# Weather App

## Project Setup

### Install all dependancies

```sh
npm install
```
or 
```sh
yarn
```

### Compile and Run for Development

```sh
npm run dev
```
or
```sh
yarn dev
```

### Compile and Build for Production

```sh
npm run build
```
or
```sh
yarn build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
